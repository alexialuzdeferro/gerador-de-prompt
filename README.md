Este projeto é o epicentro de criação de prompts para o ecossistema de **Alexia Melusine**. Um gerador de essências visuais projetado para Midjourney, Veo e outras inteligências artificiais, focado na estética **Gothic-Abyssal** e na força das entidades ancestrais.

---

## 🖤 O Aplicativo
O **Gerador de Prompts** não apenas embaralha palavras; ele conjura atmosferas. Através de uma matriz densa de variáveis, ele permite:
* **Conjuração Aleatória:** Milhares de combinações de vestimentas, cenários e iluminação chiaroscuro.
* **Eternização:** Integração direta com banco de dados Supabase (Oregon) para salvar os melhores insights.
* **Acesso Rápido:** Um arquivo dedicado para consulta e cópia instantânea das criações.

## 🛠️ Tecnologias de Conexão
Para manter este sistema respirando, utilizamos:
* **HTML5/CSS3:** Estrutura minimalista e sombria.
* **JavaScript:** Lógica de conjuração de matrizes.
* **Supabase:** Banco de dados em nuvem para memória eterna.
* **GitHub Actions:** Automação "Keep-Alive" para evitar a hibernação do sistema.
